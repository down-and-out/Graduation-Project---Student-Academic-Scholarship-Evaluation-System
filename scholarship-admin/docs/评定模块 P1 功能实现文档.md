### 1.2 业务流程更新

```
┌──────────────────────────────────────────────────────────────┐
│                    评定计算流程（更新后）                       │
└──────────────────────────────────────────────────────────────┘

课程成绩计算：
  查询学生课程成绩 → 计算加权平均分 → 应用规则上限 → 返回分数
       ↓
  学分 × 成绩 ÷ 总学分 = 加权平均分

德育表现计算：
  查询学生德育记录 → 累加各类型得分 → 应用规则上限 → 返回分数
       ↓
  志愿服务 + 社会实践 + 荣誉称号 + 学生干部 + 其他 = 总分
```

#### 2.1.3 计算公式

**加权平均分：**
```
加权平均分 = Σ(课程成绩 × 学分) ÷ Σ学分
```

**代码示例：**
```java
BigDecimal totalWeightedScore = BigDecimal.ZERO;
BigDecimal totalCredits = BigDecimal.ZERO;

for (CourseScore score : scores) {
    totalWeightedScore = totalWeightedScore.add(
        score.getScore().multiply(score.getCredit())
    );
    totalCredits = totalCredits.add(score.getCredit());
}

BigDecimal weightedAverage = totalWeightedScore.divide(
    totalCredits, 2, RoundingMode.HALF_UP
);
```

## 三、评定计算服务更新

### 3.1 更新内容

**EvaluationCalculationServiceImpl** 进行了以下更新：

1. 注入 `CourseScoreService` 和 `MoralPerformanceService`
2. 更新 `calculateCourseScore()` 方法，使用加权平均分
3. 更新 `calculateQualityScore()` 方法，使用德育总分

### 3.2 课程成绩分数计算逻辑

```java
@Override
public BigDecimal calculateCourseScore(Long studentId, Long batchId) {
    // 1. 获取评分规则
    List<ScoreRule> rules = scoreRuleService.listAvailableByRuleType(5);

    // 2. 计算加权平均分
    BigDecimal weightedAverage = courseScoreService.calculateWeightedAverage(
        studentId, batchId
    );

    // 3. 应用最高分限制
    BigDecimal maxScore = rules.stream()
        .filter(rule -> rule.getMaxScore() != null)
        .map(ScoreRule::getMaxScore)
        .min(BigDecimal::compareTo)
        .orElse(null);

    if (maxScore != null && weightedAverage.compareTo(maxScore) > 0) {
        weightedAverage = maxScore;
    }

    return weightedAverage;
}
```

### 3.3 综合素质分数计算逻辑

```java
@Override
public BigDecimal calculateQualityScore(Long studentId, Long batchId) {
    // 1. 获取德育表现总分
    BigDecimal totalScore = moralPerformanceService.calculateTotalScore(
        studentId, batchId
    );

    // 2. 应用最高分限制
    List<ScoreRule> rules = scoreRuleService.listAvailableByRuleType(6);
    BigDecimal maxQualityScore = rules.stream()
        .filter(rule -> rule.getMaxScore() != null)
        .map(ScoreRule::getMaxScore)
        .min(BigDecimal::compareTo)
        .orElse(null);

    if (maxQualityScore != null && totalScore.compareTo(maxQualityScore) > 0) {
        totalScore = maxQualityScore;
    }

    return totalScore;
}
```

```sql
CREATE TABLE `course_score` (
  `id` bigint NOT NULL AUTO_INCREMENT COMMENT '记录 ID',

  `credit` decimal(5,2) DEFAULT NULL COMMENT '学分',
  `score` decimal(5,2) DEFAULT NULL COMMENT '成绩',
  `gpa` decimal(3,2) DEFAULT NULL COMMENT '绩点',
  `academic_year` varchar(20) DEFAULT NULL COMMENT '学年',
  `semester` tinyint DEFAULT NULL COMMENT '学期：1-第一学期 2-第二学期 3-夏季学期',
  `exam_date` date DEFAULT NULL COMMENT '考试日期',
  `deleted` tinyint DEFAULT '0' COMMENT '逻辑删除：0-未删除 1-已删除',
  `create_time` datetime DEFAULT NULL COMMENT '创建时间',
```
