### 2.1 用户管理

#### 2.1.1 新增文件

| 文件 | 说明 |
|------|------|
| `SysUserService.java` | 用户服务接口 |
| `SysUserServiceImpl.java` | 用户服务实现 |
| `SysUserController.java` | 用户管理控制器 |
| `SysUserMapper.java` | 用户 Mapper（已存在） |

#### 2.1.2 API 接口

| 接口路径 | 方法 | 权限 | 功能 |
|---------|------|------|------|
| `/sys/user/page` | GET | ADMIN | 分页查询用户 |
| `/sys/user/{id}` | GET | ADMIN | 根据 ID 查询用户 |
| `/sys/user` | POST | ADMIN | 新增用户 |
| `/sys/user` | PUT | ADMIN | 更新用户 |
| `/sys/user/{id}` | DELETE | ADMIN | 删除用户 |
| `/sys/user/batch` | DELETE | ADMIN | 批量删除用户 |
| `/sys/user/reset-password/{id}` | PUT | ADMIN | 重置密码 |

#### 2.1.3 功能说明

- **分页查询**：支持按用户名、姓名、用户类型、状态筛选
- **新增用户**：自动加密密码，默认状态为正常
- **重置密码**：使用 BCrypt 加密
- **批量删除**：支持一次性删除多个用户

**加权平均分：**
```
加权平均分 = Σ(课程成绩 × 学分) ÷ Σ学分
```

### 2.5 前后端参数对齐修复

#### 2.5.1 修复的接口

| 接口 | 原方式 | 修复后 |
|------|--------|--------|
| `PUT /paper/review/{id}` | `@RequestParam` | `@RequestBody` |
| `PUT /patent/audit/{id}` | `@RequestParam` | `@RequestBody` |
| `PUT /result-appeal/handle/{id}` | `@RequestParam` | `@RequestBody` |
| `PUT /application/review/{id}` | `@RequestParam` | `@RequestBody` |

#### 2.5.2 修改示例

**修改前：**
```java
@PutMapping("/review/{id}")
public Result<Void> review(
    @PathVariable Long id,
    @RequestParam Integer status,
    @RequestParam String reviewComment) {
    // ...
}
```

**修改后：**
```java
@PutMapping("/review/{id}")
public Result<Void> review(
    @PathVariable Long id,
    @RequestBody ReviewPaperRequest request) {
    // request.getStatus()
    // request.getReviewComment()
}

// 内部记录类
public record ReviewPaperRequest(
    Integer status,
    String reviewComment
) {}
```
